# 🚀 Mini HTML Projects Collection

A collection of interactive HTML, CSS, and JavaScript applications built for fun, experimentation, and learning. This repository showcases a variety of mini projects including games, visualizations, and creative UI concepts.

## 📌 Overview

This repository contains multiple lightweight web-based applications developed to explore:

Game mechanics and logic
Interactive UI/UX design
Browser-based animations
Real-time data visualization techniques

Each project is self-contained and can be run directly in the browser.

## 🧩 Projects Included
### 🎮 Games
Cyber Siege Action Strategy Game
A fast-paced strategy game with action elements.
Pixel Conquest Strategy Game
A pixel-style strategy experience focused on tactical gameplay.
Precision Strike Sniper Game
A targeting and accuracy-based shooting game.

### 🎨 Visual & Interactive Apps
Interactive Particle System
A dynamic particle animation system with user interaction.
Kaleidoscope
A visually engaging pattern generator with symmetry effects.
Real-Time Audio Visualizer
A browser-based audio visualizer that reacts to sound input.

## 🛠️ Technologies Used
HTML5  
CSS3  
JavaScript (Vanilla JS)  
Canvas API (for animations and rendering)  

## ▶️ How to Run
Clone the repository:
git clone https://github.com/your-username/mini-html-projects.git

Open the index.html file in your browser.
